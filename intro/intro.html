Installation

For users

BMT is available on PyPI and can be installed via pip:

pip install bmt

For developers

You can install BMT from GitHub as follows:

# clone the repo
git clone https://github.com/biolink/biolink-model-toolkit

# cd into the folder
cd biolink-model-toolkit

# install
python setup.py install
